# NumericalSnippets
Java 言語向けの数値計算スニペットであるソースコードを提供する.
Java の標準APIのみを利用している.
ライブラリではないため, どの Java バージョンで動作するかは未定義である.
次が含まれる.

- `snippet.num` パッケージ
    - 三角関数の計算に関する補助
